-Berikut penjelasan mengenai tools yang digunakan dan bagaimana menjalankannya-
Tools yang digunakan :
1. Xampp
2. Web Browser 

Cara Menjalanakan :
1. Pertama, buka Xampp terlebih dahulu dan start pada Apache dan MySQL 
2. Import terlebih dahulu database.sql yang telah disediakan pada folder ke phpMyAdmin
3. Letak penyimpanan implementasi harus disimpan pada folder C:\xampp\htdocs\nama_foler 
4. Setelah itu, buka localhost pada web browser lalu pilih folder penyimpanan implementasi.
5. Lalu, pilih folder menu dan aplikasi siap dijalankan. 

Namun, ada beberapa yang perlu diketahui :
Pada saat membuka aplikasi melalui localhost, yang akan ditampilkan pertama ialah role pada Admin.
Maka, untuk membuka role dosen ataupun staf BAA, ada baiknya melihat username dan password pada role Admin.
Atau, kita bisa menambahkan pengguna baru untuk dosen dan staf BAA.
